## 快速参考

### 路径变量

许多设置支持模板变量以实现灵活的路径配置：

| 变量                 | 描述               | 示例                     |
| -------------------- | ------------------ | ------------------------ |
| `${workspace}`       | 工作区根目录       | `/home/user/project`     |
| `${tmp}`             | 系统临时目录       | `/tmp`                   |
| `${home}`            | 用户主目录         | `/home/user`             |
| `${dirname}`         | 文件目录           | `/home/user/project/src` |
| `${relativeDirname}` | 相对目录           | `src`                    |
| `${basename}`        | 带扩展名的文件名   | `main.cpp`               |
| `${basenameNoExt}`   | 不带扩展名的文件名 | `main`                   |
| `${extname}`         | 文件扩展名         | `.cpp`                   |
