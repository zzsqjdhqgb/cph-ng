# 删除测试样例

从题目中删除特定测试样例。

## 概述

删除测试样例功能允许您从题目中删除单个测试样例。这对于清理不需要的或冗余的测试样例很有用。

## 用户交互

### 触发功能

**方法：删除图标**

- 找到要删除的测试样例
- 悬停或点击测试样例以显示控件
- 点击该特定测试样例的删除/垃圾桶图标
- 无确认对话框 - 立即删除

### 前置条件

- 必须已加载题目
- 至少必须存在一个测试样例

### UI 组件

- 删除图标/按钮出现在每个测试样例内
- 通常位于测试样例标题或控件区域
- 图标颜色通常指示破坏性操作

## 内部操作

### 工作原理

1. **用户点击删除**：点击特定测试样例的删除图标
2. **从数组中删除**：从题目的测试样例数组中删除测试样例
3. **更新索引**：剩余测试样例重新编号（#1、#2 等）
4. **保存**：更新题目文件，不包含已删除的测试样例
5. **刷新 UI**：测试样例列表立即更新

### 被删除的内容

- 测试样例输入数据
- 预期答案数据
- 测试样例执行结果（如果有）
- 题目中的测试样例条目

### 被保留的内容

- 其他测试样例保持不变
- 题目元数据不变
- 源代码不变

## 无确认

与题目删除不同，测试样例删除不显示确认对话框。这允许快速清理多个测试样例。点击删除图标时要小心。

## 工作流程示例

### 删除重复的测试样例

1. 识别重复或不需要的测试样例
2. 找到其删除图标
3. 点击删除图标
4. 测试样例立即被删除
5. 剩余测试样例重新编号

### 导入后清理

导入文件中的测试样例后：

1. 查看所有导入的用例
2. 识别不必要的用例
3. 删除每个不需要的用例
4. 仅保留相关测试样例
